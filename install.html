Contraseña → Anki

One-click extractor: turns a Contraseña vocabulary lesson page into an Anki-importable TSV file.

Contraseña → Anki

Single lesson — drag to bookmarks, then click while viewing any show_hide lesson page.

Contraseña → Anki (bulk)

All lessons at once — drag to bookmarks, click on any show_hide page, accept the prefilled ID list (or edit it). Produces a single contrasena_all.tsv.

Contraseña Grammar → Anki

Conjugation tables — drag to bookmarks, then click while viewing a grammar Grammar_*.html page. Produces a TSV of "{verb} — {subject}" → conjugation cards, one section per verb.

How to use it

  1. Drag a purple button above onto your bookmarks bar (you only do this once).
  2. Open any Contraseña show_hide lesson, e.g.
    https://s3.us-east-2.amazonaws.com/contrasena/interactives/show_hide/index2.html?id=u1_01_02v2
  3. Click the bookmark.
    • Single: downloads <lesson-id>.tsv for the current page.
    • Bulk: prompts with a list of all known lesson IDs, then downloads one contrasena_all.tsv with everything.
  4. In Anki Desktop: File → Import → pick the TSV file. When prompted, choose a deck (e.g. "Spanish"). Anki will read the column types from the file header automatically.

Bulk grabber (Node script)

If you'd rather run this from a terminal, grab_all.mjs brute-forces the lesson namespace and writes per-lesson TSVs plus a combined contrasena_all.tsv:

node grab_all.mjs                       # brute-force u1..u6 (default)
node grab_all.mjs --ids ids.txt          # explicit list, one ID per line
node grab_all.mjs --units 3 --out ./out  # narrower sweep, custom dir

What ends up in Anki

  • Two fields per card: Spanish (front) and English (back).
  • Each card gets two hierarchical tags:
    • Contrasena::lessons::<lesson_id> — e.g. Contrasena::lessons::u1_01_02v2
    • Contrasena::sections::<section_name> — e.g. Contrasena::sections::los_perfiles
  • Tags appear as a browseable tree in Anki's sidebar — that's how you study "just one section."
  • If you want both Spanish→English and English→Spanish cards: in the Anki import dialog, set the note type to Basic (and reversed card).

Phone setup (one-time)

  1. Install AnkiDroid from the Play Store (free).
  2. Create a free AnkiWeb account.
  3. Sign in on Anki Desktop (Sync button) and AnkiDroid. Your deck syncs automatically.

Updating the bookmarks

If you edit bookmarklet.js, bulk_bookmarklet.js, or grammar_bookmarklet.js, also paste the new contents into the matching <script id="src"> / <script id="src-bulk"> / <script id="src-grammar"> block in this file, then reopen this page and drag the button to your bookmarks bar again (replacing the old one).
